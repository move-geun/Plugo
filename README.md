# Plugo Assignment

### 포팅 메뉴얼

+ `npm i`를 통해 실행에 필요한 라이브러리를 설치합니다
+ `json-server ./data.json --port 4000` 명령어를 입력하여 json-server를 실행시킵니다
+ `npm start`
